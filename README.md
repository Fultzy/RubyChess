  Assignment for the Odin Project.

  Running this game:
  - make sure you have ruby v-2.6.2 installed
  - run `git clone git@github.com:Fultzy/RubyChess.git`
  - then run `ruby rubychess.rb` to play the game!
  - it should be that easy!
